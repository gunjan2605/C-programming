**README**
Compiler used: GCC 
OS used: macOS Big Sur Version 11.2.2
Running Instructions: 
1. Download and save the file in appropriate folder 
2. Open terminal window 
3. Open the desired folder through the terminal window
4. To compile the program, type "gcc - o <name> scenario1.c". This command creates an executable file of the name - <name>.out 
5. To run this executable file type ./<name>
6. This should display the output in the terminal 
7. Same instructions can be followed for other .c files
